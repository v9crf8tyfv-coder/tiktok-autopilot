Automatique

Automatique est un outil personnel qui aide un créateur à produire chaque jour une courte vidéo verticale (format 9:16) à partir de sujets d'actualité, puis à la téléverser sur son propre compte TikTok.

Comment ça marche

  1. L'outil analyse les tendances publiques (Google Trends, Wikipédia).
  2. Il rédige un script factuel et génère une voix, des sous-titres et un montage.
  3. Via Login Kit, le créateur autorise l'application à accéder à son propre compte.
  4. Via la Content Posting API (scope video.upload / video.publish), la vidéo est envoyée sur le compte du créateur.

Utilisation strictement personnelle. Aucune donnée de tiers n'est collectée.
